Sobre

Jhulis é um projeto Open Source de API que valida boas práticas de modelagem de REST APIs.

Instruções

Cole no editor o seu contrato em Open API Specification (Swagger) v1, 2 ou 3.

Clique no botão Validar e analise quais regras o Jhulis retornou como sugestão de melhoria para o seu contrato.

Faça os ajustes necessários e valide novamente.

Este editor não visa ser utilizado em ambientes produtivos. O objetivo dele é ajudar o desenvolvedor a entender e testar o funcionamento do Jhulis.

Para conhecer mais sobre o Jhulis, leia a documentação da API.

Política de privacidade

Os dados das requisições podem ser armazenados por tempo limitado no servidor visando exclusivamente log para melhorias sistêmicas e análises de bug, sendo descartados em caso de não utilização. O armazenamento do contrato, assim como preferências do editor, também podem ocorrer na máquina do usuário no cache do navegador. Portanto, em caso de contratos com informações confidenciais, lembre-se de limpar o cache do navegador.
